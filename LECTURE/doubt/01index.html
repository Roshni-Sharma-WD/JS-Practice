(Simple) Javascript (WD)


Introduction

  • Javascript is a client side scripting language
  • For making page dynamic from client side
  • Event based programming language example:- on mouse click, on keyboard button press etc.
  • Loosely typed language
  • Variable doesn't need to be declared before its use
  • For Chrome We have V8 Engine to compiling the JS
  • We have " consol.log(); " is a developer option for debugging
  • We have " document.write(); " is a option for write in html document

Usage

  • We can build Slider using JS
  • We can validate a form using JS
  • We can also make a dynamic changes in HTML/CSS using JS

Ways to apply JS

mainly we can apply js two ways :

  1. by using script tag i.e. internal js
    • in head section
    • in body section
    • after end of html document
  2. by using external Js file using script tag with src attribute i.e. external js
    • in head section
    • in body section
    • after end of html document
that means you can write this internal or external js anywhere in html document i.e. in head section or body section or after the end of html document.
