Variable in JavaScript

  • variables in JavaScript are container to store the value. Value can be anything i.e. number, string, array etc.
  • There are three types of Variables in JavaScript

    1. var
    2. let
    3. const

Usage of Variables

var name; <= Declaration
name = "Parth" <= Assignment

var name = "Parth" <= Declaration and Assignment(Togather)

note: a string(text) written in commas where number wriiten without commas
If number written in commas then it is considered as a string(text).

Difference in var, let & const.

var is old Variable. which is used in old webpages before 2015.
Where let and const variables are introduced in 2015 for better Usage.

var variables can be updated and re-declared within its scope
let variables can be updated but not re-declared;
const variables can neither be updated nor re-declared

const variable needs to declare and assign togather only.
That means you cannot first declare and than assign a value.
So always write this way
i.e. 'const nameconst = "Parth"'

note: re-declaration not allowed means you can't declare second time means cant create second time with same name
re-assignment not allowed means you can't assign value second time to its variable
